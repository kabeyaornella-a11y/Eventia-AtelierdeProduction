EVENTIA SIGNATURE — LISTE DES SKILLS ET LEUR UTILITÉ

1. Eventia Core Memory

Objectif :
Conserver toute la mémoire stratégique d’Eventia Signature.

Utilité :
Permet à tous les autres Skills de connaître l’ADN de la marque, les univers fondateurs, les composants validés, les offres et les règles de création.

⸻

2. Eventia Directeur Artistique

Objectif :
Transformer un brief client en direction artistique.

Utilité :
Choisir l’univers fondateur adapté, la symbolique, l’ambiance, la palette, les typographies et la narration.

⸻

3. Eventia Symbolic System

Objectif :
Garantir la cohérence symbolique.

Utilité :
Vérifier que chaque élément présent dans une expérience possède un sens et une justification.

⸻

4. Eventia Save The Date Designer

Objectif :
Créer les Save The Date Eventia Signature.

Utilité :
Concevoir les expériences Save The Date, les ouvertures, les textes et les variantes personnalisables.

⸻

5. Eventia Save The Date Product Builder

Objectif :
Industrialiser les Save The Date.

Utilité :
Créer des modèles réutilisables et facilement personnalisables pour gagner du temps.

⸻

6. Eventia Save The Date Sales Writer

Objectif :
Vendre les Save The Date.

Utilité :
Créer les fiches produits, argumentaires commerciaux et contenus marketing.

⸻

7. Eventia Invitation Builder

Objectif :
Construire les invitations complètes.

Utilité :
Assembler les différents blocs Eventia Signature dans une expérience cohérente.

⸻

8. Eventia Storyboard Master

Objectif :
Créer le scénario de l’expérience.

Utilité :
Définir l’ordre des écrans, les révélations, les moments forts et les transitions.

⸻

9. Eventia Experience Architect

Objectif :
Concevoir le parcours utilisateur.

Utilité :
Déterminer où placer les interactions, les émotions et les appels à l’action.

⸻

10. Eventia Collection Builder

Objectif :
Créer de nouvelles collections.

Utilité :
Développer de nouveaux univers compatibles avec l’ADN Eventia Signature.

⸻

11. Eventia Asset Director

Objectif :
Déterminer les assets nécessaires.

Utilité :
Lister les illustrations, vidéos, icônes, textures et éléments à produire.

⸻

12. Eventia Illustration Director

Objectif :
Piloter les illustrations.

Utilité :
Définir le style illustratif et les éléments graphiques propres à chaque collection.

⸻

13. Eventia Site Architect

Objectif :
Piloter le site Eventia Signature.

Utilité :
Créer et optimiser les pages, l’arborescence, les parcours et la conversion.

⸻

14. Eventia Site Copywriter

Objectif :
Rédiger les contenus du site.

Utilité :
Créer les textes des pages, offres, collections et appels à l’action.

⸻

15. Eventia Product Manager

Objectif :
Développer l’offre Eventia Signature.

Utilité :
Imaginer de nouveaux produits, services, options et stratégies de croissance.

⸻

16. Eventia Design Guardian

Objectif :
Contrôler la cohérence visuelle.

Utilité :
Détecter les erreurs de design, les incohérences et les éléments qui dégradent le rendu premium.

⸻

17. Eventia Experience Auditor

Objectif :
Auditer une expérience terminée.

Utilité :
Évaluer la qualité globale de l’invitation avant livraison.

⸻

18. Eventia Luxury Validator

Objectif :
Valider le niveau premium.

Utilité :
Déterminer si une création correspond réellement à un positionnement haut de gamme.

⸻

19. Eventia Moodboard Director

Objectif :
Créer les moodboards.

Utilité :
Définir les inspirations, palettes, typographies et directions visuelles avant création.

⸻

20. Eventia Invitation Critic

Objectif :
Critiquer une invitation.

Utilité :
Identifier ce qui fonctionne, ce qui est faible et ce qui doit être amélioré.

⸻

21. Eventia Client Brief Analyzer

Objectif :
Analyser les réponses du client.

Utilité :
Transformer un questionnaire ou un échange client en cahier des charges exploitable.
